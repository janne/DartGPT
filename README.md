# DartGPT

## Setup

- `dart pub get`
- `dart compile exe dart_gpt.dart -o gpt`
- `mv gpt <DIRECTORY IN YOUR PATH>`

## Usage

```
$ cat "Hey there!"|gpt
```

or

```
$ gpt
Hey there!<Ctrl+D>
```
